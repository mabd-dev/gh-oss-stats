# Release Notes: v0.3.2

Release Date: December 31, 2025
Codename: Command Architecture & Badge JSON Processing

## Major Features

### Sub-Command Architecture [#24](https://github.com/mabd-dev/gh-oss-stats/pull/24)

Redesigned CLI to support sub-commands for different use cases. The tool now provides a clean separation of concerns with three distinct command modes:

- **`default` command** - Fetch contribution stats from GitHub (main functionality)
  - Unchanged user-facing behavior
  - Refactored internal command structure for maintainability

- **`badge` sub-command** - Generate badges from existing stats JSON data
  - Process previously saved stats without re-fetching from GitHub
  - Avoid GitHub API rate limits when generating multiple badge variants
  - Supports two input methods:
    - `--from-file <path>` - Read stats from JSON file
    - `--data <json>` - Inline JSON string input
  - All badge configuration flags available (style, theme, variant, etc.)

- **`demo` sub-command** - Generate demo badges with sample data
  - Preview badge styles and themes without hitting GitHub API
  - Useful for testing badge designs and documentation
  - All badge configuration flags available
  - Built-in realistic sample data
